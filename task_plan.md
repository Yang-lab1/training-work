# 任务计划：Interview OS 个人面试训练 MVP

## 目标
把 Interview OS 砍成真实可用的个人面试训练 MVP：不再做虚拟上传、假流程、复杂导航和完整求职 OS，只保留真实文件上传、真实浏览器录音、真实今日训练任务保存。

## 已做决策
| 决策 | 理由 |
|------|------|
| 不再沿用 workflow | 当前目标是个人可用 MVP，不是完整求职系统 |
| 用 IndexedDB 保存录音 Blob | localStorage 不适合保存音频二进制 |
| 文件只保存在本机浏览器，不上传服务器 | 本版没有后端，避免假装上传到云端 |
| 每个训练任务都可单独录音、重练、勾选 | 满足当天训练记录的真实使用场景 |

## 验收状态
- 用户打开网站即可上传 CV。
- 用户可直接录中文自我介绍、英文自我介绍、Miro 项目讲解。
- 录音可回放、下载。
- 今日训练任务可勾选并保存到本地浏览器。
- 页面不再展示复杂导航和完整求职 OS。
